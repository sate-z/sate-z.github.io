paramiko 遵循SSH2协议，支持以加密和认证的方式，进行远程服务器的连接。可以用来远程登录主机并执行命令，传输文件等。

远程连接方法

使用paramiko模块有两种连接方式，一种是通过paramiko.SSHssh()函数，另外一种是通过paramiko.Transport()函数。
